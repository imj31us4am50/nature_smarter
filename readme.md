
# Nature Smarter Project
![Logo](https://imj31us4am50.github.io/imgs/ns-logo.png)

Interactive website created to raise awareness about the dangers wildlife is facing. All media (videos and texts) were manually sourced from copyright-free websites for educational and informational purposes.


## Team

- [@imj31us4am50](https://github.com/imj31us4am50) - front-end & content
- [@AvramAlexandruV](https://github.com/AvramAlexandruV) - back-end & user interaction


## Features

- Responsive layout for all devices
- Intuitive navigation throughout the platform
- User login and registration
- Account management for users
- Data protection using SHA-2 encryption
- User interaction with content (likes and comments)
- Question submission on the About page
